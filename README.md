# Superstore Sales Dashboard (Power BI)

## 📊 Overview
This project analyzes sales performance using Power BI.

## 🚀 Features
- KPI metrics: Sales, Profit, Profit Margin
- Region-wise and category analysis
- Interactive filters (slicers)

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
